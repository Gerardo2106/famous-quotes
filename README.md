# Famous Quotes App

## Description

A server-rendered web application that dynamically displays a collection of quotes using a Node.js backend.

## Key Contributions

* Developed backend functionality using Express.js for routing and content delivery
* Implemented EJS templating to dynamically render quote data
* Designed a clean and user-friendly interface with CSS
* Organized project structure for scalability and readability

## Tech Stack

Node.js, Express.js, EJS, CSS

## Installation

```bash
git clone https://github.com/Gerardo2106/famous-quotes.git
cd famous-quotes
npm install
```

## Usage

```bash
node index.mjs
```

Visit: http://localhost:3000

## Project Structure

* views/ – Templates
* public/ – Static files
* index.mjs – Server entry point
